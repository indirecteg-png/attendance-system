# نظام الحضور والانصراف - نسخة بسيطة

نظام متكامل لإدارة الحضور والانصراف للموظفين يعمل محلياً ويمكن رفعه على GitHub Pages للاستخدام من أي مكان في العالم.

## 🚀 المميزات

### 📱 **واجهة مستخدم حديثة**
- تصميم متجاوب يعمل على جميع الأجهزة
- دعم كامل للغة العربية مع RTL
- واجهة سهلة الاستخدام وبديهية
- يعمل بدون إعداد معقد

### ⏰ **إدارة الحضور**
- تسجيل الحضور والانصراف
- حساب الأوفر تايم تلقائياً
- نظام الاستراحات
- عرض الحالة اليومية
- حساب ساعات العمل

### � **إدارة الموظفين**
- إضافة موظفين جدد
- تعديل بيانات الموظفين
- حذف الموظفين
- تصنيف حسب الأقسام

### � **إدارة البيانات**
- تصدير البيانات كملف JSON
- استيراد البيانات من ملف JSON
- مسح جميع البيانات
- نقل البيانات بين الأجهزة

## 🚀 **التشغيل المحلي**

1. **فتح التطبيق**
   - افتح ملف `simple-local.html` في أي متصفح حديث
   - أو استخدم خادم محلي:
   ```bash
   # باستخدام Python
   python -m http.server 8000
   
   # أو باستخدام Node.js
   npx serve .
   ```

2. **تسجيل الدخول**
   - اسم المستخدم: `admin`
   - كلمة المرور: `123456`

## 🌐 **النشر على GitHub Pages**

لرفع النظام على GitHub Pages واستخدامه من أي مكان في العالم:

1. **أنشئ حساب GitHub** على [github.com](https://github.com/)
2. **أنشئ مستودع جديد** (New Repository)
3. **ارفع الملفات التالية:**
   - simple-local.html
   - script-simple.js
   - styles.css
4. **فعّل GitHub Pages** في إعدادات المستودع
5. **استخدم النظام** من الرابط المُنشأ

**للتفاصيل الكاملة، راجع ملف `GITHUB_DEPLOYMENT_GUIDE.md`**

## 💾 **إدارة البيانات**

### تصدير البيانات
1. سجل الدخول في النظام
2. اذهب إلى "إدارة البيانات"
3. اضغط "تصدير"
4. سيتم تحميل ملف JSON

### استيراد البيانات
1. اذهب إلى "إدارة البيانات"
2. اختر ملف JSON
3. اضغط "استيراد"

### نقل البيانات بين الأجهزة
1. صدر البيانات من الجهاز 1
2. أرسل الملف عبر البريد الإلكتروني أو Google Drive
3. استورد البيانات على الجهاز 2

## 🛠️ **التقنيات المستخدمة**

- **HTML5** - هيكل الصفحات
- **CSS3** - التنسيقات مع Tailwind CSS
- **JavaScript ES6+** - منطق التطبيق
- **Font Awesome** - الأيقونات
- **LocalStorage** - تخزين البيانات

## 📊 **بيانات التطبيق**

يتم تخزين جميع البيانات في LocalStorage بالمتصفح:

- بيانات الموظفين
- سجلات الحضور والانصراف
- بيانات المستخدمين

يمكن تصدير البيانات واستيرادها يدوياً للنسخ الاحتياطي والنقل بين الأجهزة.

## 🐛 **استكشاف الأخطاء**

### مشاكل شائعة

1. **لا يعمل تسجيل الدخول**
   - تأكد من استخدام البيانات الصحيحة: admin / 123456
   - تحديث الصفحة (Ctrl+F5)

2. **لا تظهر البيانات**
   - تحقق من LocalStorage في المتصفح
   - امسح الكاش وأعد تحميل الصفحة
   - استورد البيانات من ملف JSON

3. **مشاكل في اللغة العربية**
   - تأكد من دعم RTL في المتصفح
   - تحقق من إعدادات الخطوط
